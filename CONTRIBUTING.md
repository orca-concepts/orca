# Contributing to Orca

Thanks for your interest in contributing to Orca!

**Please open an issue before submitting a pull request.** This helps avoid duplicate work and lets us discuss approach before you spend time on an implementation. For bugs, use the bug report template; for new features, use the feature request template.
